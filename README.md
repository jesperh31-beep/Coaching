# Floorball Coach

Simple site for planning træning og kampe. Denne commit tilføjer funktioner: fritekst-øvelser, mulighed for grupper per øvelse, omrokering af øvelser, billedmodal og forbedret printlayout.

Hvordan kører du lokalt:
- Åbn index.html i din browser.

Instruktioner til deploy:
- Forbind dette repo til Vercel eller Netlify og deploy fra main.
